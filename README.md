## Tecnologias

 - React Native
 - Redux
 - Redux Saga
 - JSON Server
 - Axios
 - Styled Components
 - Reactotron Redux/Saga
 - React-navigate
 - Polished
 - Immer

## Run

 - yarn
 - react-native start
 - json-server json-server.json -p 3333 -w
 - react-native run-android || run-ios
